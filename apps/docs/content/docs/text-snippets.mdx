## Was sind Text Snippets?

Text Snippets ersetzen ein kurzes Kuerzel durch einen laengeren Textbaustein. Damit sparen Sie Zeit bei wiederkehrenden Formulierungen.

## Verwendung

1. Kuerzel in ein Eingabefeld schreiben (z. B. `mfg`)
2. `Shift + F2` druecken
3. Kuerzel wird durch den hinterlegten Text ersetzt

## Gute Beispiele

| Kuerzel | Ergebnis |
|---|---|
| `anam` | `Anamnese: ...` |
| `bef` | `Befunde: ...` |
| `proc` | `Procedere: ...` |
| `mfg` | `Mit freundlichen Gruessen` |

## Best Practices

- Kuerzel kurz und eindeutig halten (2-5 Zeichen)
- Einheitliche Namenslogik verwenden (z. B. `diag`, `ther`, `proc`)
- Kollisionen mit normalen Woertern vermeiden
- Veraltete Snippets regelmaessig entfernen

## Unterschied zu Templates

- **Text Snippets:** reine Textersetzung fuer einzelne Bausteine
- **Templates:** komplette Dokumentstruktur mit dynamischen Tags

Wenn Sie strukturierte klinische Dokumente erzeugen wollen, nutzen Sie primaer [Templates](/templates).
